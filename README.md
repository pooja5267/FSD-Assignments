# 📚 Assignment Details

## Assignment 1 – Basic HTML Portfolio

### Problem Statement

Create a basic **portfolio webpage using HTML** to display personal information and basic web structure.

### Objective

* To understand the basic structure of HTML.
* To create a simple webpage layout.
* To display content using HTML elements.

### Explanation

This assignment demonstrates the use of fundamental HTML tags such as:

* `html`
* `head`
* `title`
* `body`
* `h1, h2`
* `p`
* `img`
* `a`

The portfolio webpage includes basic information such as introduction, skills, and other details.

### Output

Output screenshots of the webpage are stored inside the **Assignment-1 folder**.

## Assignment 2 – React Portfolio Application

### Problem Statement

Develop a **portfolio website using React.js** to understand component-based frontend development.

### Objective

* To learn the basic structure of a React project.
* To understand the use of **src and public folders**.
* To run and manage a React application using **npm**.
* To build a simple portfolio website using React.

### Explanation

The React project was created using **Create React App**.

Main components of the project include:

* **public folder**

  * Contains `index.html` and static assets.

* **src folder**

  * Contains the main React components and application logic.

* **node_modules**

  * Stores project dependencies installed through npm.

The application is executed using:

```
npm start
```

### Output

Output screenshots are stored in the **Outputs folder** of Assignment-2.
