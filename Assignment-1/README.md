## HTML Tags Covered

### Document Structure
- `<html>` - Root element of HTML document
- `<head>` - Contains metadata and document title
- `<title>` - Defines browser tab title
- `<body>` - Contains visible webpage content

### Text Formatting
- `<h1>` to `<h6>` - Heading hierarchy
- `<p>` - Paragraph text
- `<br>` - Line breaks

### Media & Links
- `<img>` - Image display
- `<a>` - Hyperlinks and navigation

### Data Presentation
- `<table>`, `<tr>`, `<th>`, `<td>` - Tabular data
- `<ul>`, `<ol>`, `<li>` - Lists (unordered and ordered)

### Layout & Structure
- `<div>` - Generic container elements
- `<span>` - Inline text styling

### Forms & Interaction
- `<form>` - User input collection
